# ⚡ APC Intelligence Dashboard  
### AI-Based Auxiliary Power Consumption Optimization  
**Hindalco Industries Ltd. — Renusagar Power Division (RPD)**

---

## 📌 Overview

The **APC Intelligence Dashboard** is an AI-driven analytics system designed to optimize **Auxiliary Power Consumption (APC)** in a **716 MW captive thermal power plant**.

It combines:
- 📊 Interactive Dashboard (Chart.js)
- 🤖 Machine Learning Models (LSTM, GRU, XGBoost)
- ⚙️ Equipment-Level Analysis
- 💡 Optimization Engine
- 📈 Forecasting & Decision Support

---

## 🎯 Objectives

- Reduce APC from ~8.11% to ~6.92%
- Identify high-consumption equipment (BFP, CW Pumps, Fans)
- Enable real-time decision support
- Achieve ₹50+ Crore annual savings
- Improve plant efficiency

---

## 🚀 Features

### 📊 Dashboard
- APC trend (Actual vs Forecast vs Target)
- KPI cards (APC %, Savings, Equipment contribution)
- Alerts for abnormal performance

### ⚙️ Equipment Analytics
- Equipment-wise APC contribution:
  - BFP
  - CW Pump
  - ID / FD / PA Fans
  - Mills
- Monthly trend tracking

### 🏭 Unit Performance
- 10-unit comparison
- APC vs PLF analysis
- Identify inefficient units
